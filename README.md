# browsynth

a browser-based synth. no install, just open and play.

## features

- polyphonic synth with multiple waveforms (sine, square, saw, triangle)
- wavetable mode with a cool 3D editor
- supersaw-ready unison with stereo spread & blend
- portamento/glide for acid lines
- filter, envelope (ADSR), LFO, noise, FM
- effects: reverb, tempo-synced ping-pong delay (dotted 8th!), chorus, drive, 3-band EQ, compressor
- trance gate (16-step rhythmic gate synced to BPM)
- sidechain pump + built-in four-on-the-floor kick to jam over
- global BPM that delay, gate, pump, kick and Cthulhu all sync to
- arpeggiator with different patterns
- **microtonal support!** 12, 13, 17, 24-tone scales via IPFS tuning files
- custom .tun file import
- visualizer (lissajous is beautiful btw)
- MIDI input support
- record & export to WAV
- save/load ur own presets (+ new trance factory presets: supersaw lead, gated pad, acid 303, hoover...)
- keyboard mapping for playing with ur computer keyboard

## usage

just open it in ur browser and click "Let's Browsynth!"

use ur keyboard to play notes. check the piano to see what keys do what.
